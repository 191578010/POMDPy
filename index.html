POMDPy

A framework for implementing discrete or continuous POMDPs in Python. Monte-Carlo Tree Search is used as the default POMDP solver (modeled after the popular POMCP solver).

View on GitHub Download .zip Download .tar.gz

This open-source project contains a framework for implementing discrete or continuous POMDPs in Python. The organization of the code was inspired by TAPIR and the POMCP algorithm.

What the heck is a POMDP?

Here's David Silver and Joel Veness's paper on POMCP, a ground-breaking POMDP solver. Monte-Carlo Planning in Large POMDPs

This is project has been conducted strictly for research purposes. If you would like to contribute to POMDPy or if you have any comments, suggestions, or encounter any bugs (I am sure there are still quite a few as this is a work-in-progress), feel free to send me a pull request or send me an email at pemami@ufl.edu.

Installation

From the root directory of the project, run python setup.py install

Running a sample

After tweaking the configuration files found in the config package , run the RockSample Problem with

python src/run_pomdp.py

Dependencies

This project uses:

  • Python 2.7.9
  • numpy 1.9.2

Optional, for extended functionality:

  • matplotlib 1.4.3
  • pytest 2.7.0
  • pygame 1.9.1
