POMDPy

A framework for implementing discrete or continuous POMDPs in Python. Monte-Carlo Tree Search is used as the default POMDP solver (modeled after the popular POMCP solver).

Download .zip Download .tar.gz View on GitHub

This open-source project contains a framework for implementing discrete or continuous POMDPs in Python. The organization of the code was inspired by TAPIR and the POMCP algorithm.

This is project has been conducted strictly for research purposes. If you would like to contribute to POMDPy or if you have any comments or suggestions, feel free to send me a pull request or send me an email at pemami@ufl.edu.

Belief Tree structure

Parent BeliefNode -> ActionMapping -> ActionMappingEntry -> ActionNode -> ObservationMap -> ObservationMappingEntry -> Child BeliefNode

Dependencies

This project uses:

  • Python 2.7.9
  • numpy 1.9.2
  • matplotlib 1.4.3
  • pytest 2.7.0
